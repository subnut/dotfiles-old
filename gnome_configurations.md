Shortcuts:
-----------
::Super+D		Hide all normal windows
::Ctrl+F11		Toggle Fullscreen mode

Custom
::Super+E		Command: Nautilus

In Power settings, assign Power Button to Shutdown instead of Suspend

-----------------------
Built-in
:: Applications Menu
:: Launch new instance
:: Places Status Indicator

Custom
